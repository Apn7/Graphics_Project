\section{Abstract}
% =============================================================

This report describes an interactive 3D library simulation built in C++ with Modern OpenGL 3.3.
The scene occupies a 24$\times$20$\times$6 unit room containing bookshelves, reading tables, chairs, nine ceiling fans, six pendant lamps, a librarian desk, a decorative globe, and Bezier-generated vases.
Rendering is driven by four interchangeable shading modes (flat Phong, simple texture, vertex-blend, fragment-blend), a Phong illumination model with directional, point, and spotlight sources, and frame-rate-independent animation for fans, globe, door, and windows.
User interaction is provided through first-person and bird's-eye camera modes, per-light keyboard toggles, and day/night preset switching.

% =============================================================
\section{Introduction}
% =============================================================

This project implements a fully interactive library environment to demonstrate the CSE 4208 mandatory topics within one cohesive real-time application.
The scene covers: multiple textured curved surfaces (UV sphere, ruled-surface cone, half-torus, Bezier vase), vertex and fragment texture blending, Phong lighting with directional, point, and spotlight sources, first-person and bird's-eye projection modes, $\Delta t$-based fan animation, and kinematic door/window motion.
Dependencies are GLFW (windowing), GLAD (OpenGL loading), GLM (mathematics), and \texttt{stb\_image} (textures).
The codebase separates window/input/shader utilities, renderer primitives, and scene-construction logic into three independent layers.

\section{Theoretical Background}
% =============================================================

\subsection{Model--View--Projection Transformation}

Every vertex in the scene passes through the model--view--projection (MVP) chain before reaching clip space:
\begin{equation}
    \mathbf{p}_{\text{clip}} = \mathbf{P}\,\mathbf{V}\,\mathbf{M}\,\mathbf{p}_{\text{model}}
    \label{eq:mvp}
\end{equation}
where $\mathbf{M}$ is the per-object model matrix constructed as $\mathbf{M}=\mathbf{T}\mathbf{R}\mathbf{S}$, $\mathbf{V}$ is the camera view matrix derived from position and Euler angles, and $\mathbf{P}$ is either a perspective frustum matrix or an orthographic matrix depending on the current view mode.

Surface normals cannot be transformed by $\mathbf{M}$ directly when non-uniform scaling is present. The correct transformation is:
\begin{equation}
    \mathbf{n}_{\text{world}} = \bigl(\mathbf{M}^{-1}\bigr)^{\!\top}\,\mathbf{n}_{\text{model}}
    \label{eq:normal}
\end{equation}
This inverse-transpose form guarantees that normals remain perpendicular to their surfaces after stretching, which is critical for correct Phong shading on chair legs, shelf panels, and similar thin objects.

\subsection{Phong Reflection Model}

The radiance accumulated at a fragment from $N$ point lights is:
\begin{equation}
    L = L_{\text{ambient}} + \sum_{i=1}^{N} \frac{L_{d,i}\max(\hat{n}\cdot\hat{l}_i,0) + L_{s,i}\max(\hat{r}_i\cdot\hat{v},0)^{p}}
        {k_c + k_l\,d_i + k_q\,d_i^2}
    \label{eq:phong}
\end{equation}
where $d_i$ is the fragment-to-light distance; $k_c$, $k_l$, $k_q$ are attenuation coefficients; $\hat{n}$ is the surface normal; $\hat{l}_i$ the light direction; $\hat{r}_i$ the perfect reflection; $\hat{v}$ the view direction; and $p$ the shininess exponent. Ambient light is accumulated globally using a separate strength term so it can be toggled independently.

\subsection{Cubic Bezier Curves and Surface of Revolution}

A cubic Bezier profile curve is parameterised by four control points and $t\in[0,1]$:
\begin{equation}
    \mathbf{B}(t) = (1{-}t)^3\mathbf{P}_0 + 3(1{-}t)^2t\,\mathbf{P}_1
                    + 3(1{-}t)t^2\,\mathbf{P}_2 + t^3\mathbf{P}_3
    \label{eq:bezier}
\end{equation}
Sampling $\mathbf{B}(t)$ in the radius-height plane and revolving each sample through $[0,2\pi]$ around the $Y$-axis generates a closed surface of revolution. Profile tangents, computed from the derivative $\mathbf{B}'(t)$, are used to construct smooth per-vertex normals on the resulting mesh.

\subsection{Ruled Surfaces and Swept Primitives}

A ruled surface interpolates linearly between two generator curves. The cone in this project is generated from a base circle to a degenerate apex, making each triangle a linear edge of the ruled surface. The half-torus (globe meridian ring) is produced by sweeping a circular cross-section along a semi-circular spine, resulting in a parametric surface with smoothly varying normals across 180$^\circ$.

\subsection{Vertex Format and Primitive Geometry}

All meshes share one vertex format: 3-component position, 3-component normal, and 2-component UV coordinate. This consistent layout allows all four shader programs to use the same attribute binding points. The cube (24 vertices, 36 indices with per-face normals) is the most reused primitive, while the sphere, cone, half-torus, and Bezier vase are generated procedurally and stored as \texttt{CurvedObject} instances with individually owned meshes.

\section{Bezier Surface of Revolution}
% =============================================================

\subsection{Theory}

The vase is generated by revolving a cubic Bezier profile (Eq.~\ref{eq:bezier}) around the Y-axis. Control points $\mathbf{P}_0\dots\mathbf{P}_3$ are chosen in the radius-height plane to reproduce a classic bulging vase silhouette with a widened base, narrowed neck, and flared lip. Sampling the profile at $m$ equally spaced $t$-values and rotating each sample through $n$ equally spaced angles $\theta\in[0,2\pi]$ yields an $m{\times}n$ surface grid. Neighbouring quads are triangulated to produce the final index list, and normals are derived from the cross product of the radial tangent and the profile tangent at each sample point.

\subsection{Implementation}

\begin{lstlisting}[language=C++, caption={Core Bezier point evaluation and revolved mesh sketch}]
glm::vec2 BezierPt(float t, glm::vec2 p0, glm::vec2 p1,
                             glm::vec2 p2, glm::vec2 p3) {
    float u = 1.0f - t;
    return u*u*u*p0 + 3*u*u*t*p1 + 3*u*t*t*p2 + t*t*t*p3;
}

// For each profile step i and radial slice j:
//   q   = BezierPt(t_i, ...)          // (radius, height)
//   x   = q.x * cos(theta_j)
//   y   = q.y
//   z   = q.x * sin(theta_j)
// Normals derived from cross(dtheta, dt) at each sample.
\end{lstlisting}

\section{Real-Time Animation and Interaction}
% =============================================================

\subsection{Fan and Globe Rotation}

Both the ceiling fans and the desk globe are animated using frame-rate-independent integration:
\begin{equation}
    \theta_{k+1} = \theta_k + \omega\,\Delta t
    \label{eq:fanupdate}
\end{equation}
All nine fans share a common angle accumulator, so they remain in phase. Each fan's blade transforms are recomputed every frame from its stored centre position, preventing drift.

\subsection{Door and Window Kinematics}

The door and window shutters animate toward their open or closed target angles using a clamped incremental update:
\begin{equation}
    \theta_{k+1} = \theta_k + \operatorname{sgn}(\theta^{*}-\theta_k)\,
    \min\!\bigl(\omega\,\Delta t,\;\lvert\theta^{*}-\theta_k\rvert\bigr)
    \label{eq:doorupdate}
\end{equation}
The \texttt{min} term prevents overshoot when the object approaches its destination. Door maximum open angle is 110$^\circ$ and each window shutter opens to 85$^\circ$.

\section{Challenges and Solutions}
% =============================================================

\begin{description}[leftmargin=0pt]

    \item[\textbf{Non-uniform scaling distorting normals:}]
        Thin cuboid furniture elements (shelf panels, chair legs) distort surface normals under direct $\mathbf{M}$ transformation.\\
        \textit{Solution:} All vertex shaders compute the normal matrix as $(\mathbf{M}^{-1})^\top$ to restore geometric correctness.

    \item[\textbf{Spotlight leaking below the desk:}]
        Without shadow geometry, the cone-shaped spotlight illuminated floor fragments below the opaque tabletop.\\
        \textit{Solution:} A smooth elliptical mask in the fragment shader restricts spotlight contribution to a patch centred on the tabletop surface.

    \item[\textbf{Globe stand realism:}]
        An initial blocky C-arch stand looked unconvincing and broke immersion.\\
        \textit{Solution:} A \texttt{CreateHalfTorus} generator was implemented to produce the curved 180$^\circ$ meridian ring, and the globe was elevated 22\% of a unit to expose the vertical support rod.

    \item[\textbf{Fan/lamp center conflict:}]
        The original centre pendant lamp (X$=0$, Z$=0$) clipped the center fan motor housing.\\
        \textit{Solution:} The ceiling layout was redesigned into the FLFLF column pattern, placing all six lamps at X$\in\{-4,+4\}$ so no lamp ever shares a column with any fan.

    \item[\textbf{Room felt confined:}]
        The original 16$\times$14 room left little open space after furniture placement.\\
        \textit{Solution:} Room dimensions were expanded to 24$\times$20 by changing two constants; all furniture X-positions were correspondingly spread from $\pm4.5$ to $\pm7.5$.

\end{description}

\section{Conclusion}
% =============================================================

This project demonstrates a complete Modern OpenGL 3.3 interactive application — from indexed mesh rendering and GLSL shading through texture mapping, multi-source Phong lighting, procedural curved geometry, and frame-rate-independent animation. The library scene satisfies the CSE 4208 mandatory topics: three varieties of texture mode, directional and point lighting with per-component toggles, a bird's-eye orthographic view, fan rotation, kinematic door and window motion, and curved objects generated from UV sphere, ruled-surface cone, half-torus, and Bezier surface of revolution primitives.

Future work includes a recursive fractal tree plant inside the entrance vases, shadow mapping for realistic light blocking, a four-viewport presentation mode, and a custom \texttt{rotate()} implementation independent of \texttt{glm::rotate()}.

\section{Representative Shader Excerpts}
% =============================================================

\begin{lstlisting}[language=GLSL, caption={Vertex shader: MVP transform and normal-matrix correction}]
#version 330 core
layout(location=0) in vec3 aPos;
layout(location=1) in vec3 aNormal;
layout(location=2) in vec2 aTexCoord;

uniform mat4 u_Model, u_View, u_Projection;

out vec3 v_FragPos; out vec3 v_Normal; out vec2 v_TexCoord;

void main() {
    vec4 world  = u_Model * vec4(aPos, 1.0);
    v_FragPos   = vec3(world);
    v_Normal    = mat3(transpose(inverse(u_Model))) * aNormal;
    v_TexCoord  = aTexCoord;
    gl_Position = u_Projection * u_View * world;
}
\end{lstlisting}

\begin{lstlisting}[language=GLSL, caption={Fragment shader: desk spotlight with surface mask}]
float DeskLampMask() {
    const vec2 centre = vec2(6.45, 5.95);
    const vec2 radius = vec2(1.55, 0.82);
    vec2 local        = (v_FragPos.xz - centre) / radius;
    float topMask     = smoothstep(0.88, 0.97, v_FragPos.y);
    return max(1.0 - dot(local, local), 0.0) * topMask;
}

vec3 CalcSpotLight(vec3 norm, vec3 viewDir, vec3 baseColor) {
    vec3  dir       = normalize(u_SpotLightPos - v_FragPos);
    float theta     = dot(dir, normalize(-u_SpotLightDir));
    float eps       = u_SpotCutoff - u_SpotOuterCutoff;
    float intensity = clamp((theta - u_SpotOuterCutoff)/eps, 0.0, 1.0)
                    * DeskLampMask();
    return intensity * baseColor;
}
\end{lstlisting}

\medskip
This project develops an interactive 3D Library environment using Modern OpenGL 3.3. The library represents a realistic indoor reading space containing bookshelves, reading tables, chairs, windows, and lighting elements. The objective is to demonstrate core computer graphics concepts — 3D modelling, transformations, lighting, texture mapping, and camera navigation — in a structured and explainable manner. The scene is rendered in real-time using a custom OpenGL pipeline with GLSL vertex and fragment shaders.

\medskip\noindent\textbf{Proposed Features}

\begin{itemize}[leftmargin=1.5em]
    \item Complete 3D library interior: walls, floor, ceiling, bookshelves, reading tables, chairs.
    \item Bookshelves populated with books modelled from scaled and repeated primitives.
    \item Smooth first-person camera navigation (WASD + mouse) and bird's-eye view.
    \item Interactive lighting: directional sunlight, warm pendant point lights, desk spotlight.
    \item Day and night modes by adjusting directional light intensity and colour.
    \item Nine ceiling fans animated using real-time $\Delta t$-based angular integration.
    \item Phong shading with ambient, diffuse, and specular per-component toggles.
    \item Texture mapping on floor, walls, furniture, and door surfaces.
    \item Curved objects: UV sphere (globe), ruled-surface cone, half-torus, Bezier vase.
    \item Interactive door and window kinematics.
\end{itemize}
